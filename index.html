Project Prometheus

Project Prometheus is centered around the incredible Raspberry Pi 2. The idea consists of a plug and play device that can automatically back up connected drives to Amazon Web Services such as Amazon Glacier and Amazon S3. Allowing the speed and size of onsite storage, with the assurance of an off-site back-up.

Sponsored by WDLabs, the project is in its inital stages.

Updates: Raspberry Pi set up with raspbian and ssh. Auto mounted USB configured. AWSCLI Installed.

To find out more information about this project, check out the github.
